About This Website

This recipe website was created as a final project to demonstrate HTML and CSS skills. It includes a clean layout, styled components, and a simple navigation system.

Features showcased in this project include:

  • Consistent layout using Flex/Grid
  • Reusable design with external CSS
  • Recipe cards that link to full instructions
  • A responsive layout suitable for mobile and desktop
  • Use of the box model (padding, margin, border)

The goal of this project is to provide simple and delicious recipes while demonstrating college-level web design concepts in an organized and visually appealing format.
